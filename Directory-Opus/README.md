# Directory Opus 常用快捷键

个人常用的 Directory Opus 快捷键记录。

## 双栏操作

| 快捷键          | 功能                                  |
| ------------ | ----------------------------------- |
| `Alt+V`      | 打开/关闭双栏模式                           |
| `Tab`        | 切换活动栏                               |
| `Alt+B`      | 交换左右双栏                              |
| `Ctrl+Enter` | 在另一侧栏中打开当前选中目录                      |
| `Ctrl+1`     | 复制到目标窗格（Copy Files），双栏模式下无需手动指定目标目录 |
| `Ctrl+2`     | 移动到目标窗格（Move Files），双栏模式下无需手动指定目标目录 |

## 标签页管理

| 快捷键            | 功能         |
| -------------- | ---------- |
| `Ctrl+T`       | 新建标签页      |
| `Ctrl+Shift+T` | 恢复最近关闭的标签页 |
| `Shift+H`      | 切换到左侧标签页   |
| `Shift+L`      | 切换到右侧标签页   |

## 路径导航

| 快捷键      | 功能     |
| -------- | ------ |
| `Ctrl+L` | 定位到路径栏 |

## 常用工作流

### 双栏复制/移动文件

1. `Alt+V` 打开双栏模式。
2. 使用 `Ctrl+Enter` 在另一侧栏中打开目标目录（可选）。
3. 在当前窗格中选中文件或文件夹。
4. 按 `Ctrl+1` 将文件复制到目标窗格，或按 `Ctrl+2` 将文件移动到目标窗格，无需切换活动栏或手动指定目标目录。

### 标签页操作

1. `Ctrl+T` 新建标签页。
2. `Shift+H` / `Shift+L` 切换标签页。
3. `Ctrl+Shift+T` 恢复误关闭的标签页。

### 快速定位

1. `Ctrl+L` 聚焦路径栏。
2. 输入路径后回车跳转。
